  • CASE DoesMatter


    Rules of thumb:

    • Constants are ALL CAPS for example "const = ALLUPPERCASE"

  • Data types


    • String - letters and symbols, typically words and sentences
    * Note: Use a quote in a string - See JS example 1
    • Boolean - Binary TRUE / FALSE "or" 1 / 0
    • Null - Stores the intentional absence of a value
    • Undefined - When you set a variable but don't set it to anything
    - To find data type - in console type console.log(typeof variableName);

  • Shorthand Variables


    * Note: NaN - Not a number. For Example:
    a = 4;
    b = "computer";
    a + b = NaN

    a - a + 1; can be typed as: a++;
    a = a + 4; can be typed as: a += 4;

  • Conditional Statements & Logic


    if (some condition) {
    Do something. When condition is true
    } else {
    Do something else When condition is false
    }
    * Note - For if variables are the same use x2 =! For example: if (a == b) NOT if (a = b)
    * Note - For absolute identical testing to see if 2 variables are the same for example var a = 2; and var b = "2"; then use x3 = for example: if (a === b)
    • If a value is less than "or" greater than - if (a < b) "or" if (a > b)
    • If a value is less than and equal to "or" greater than and equal to - if (a <= b) "or" if (a >= b)
    • If a value is NOT equal to - if (a != b) shorthand - if ( !a ) { }
    • If a value is NOT identical to - if (a !== b)

  • Arrays


    • Arrays are used to store variables in a comma separated list • Arrays are wrote as: var MyTestArray = ["apples", "bananas", "pears", "oranges"];
    • Arrays can contain whatever you like ie. numerical data, strings, booleans etc.
    • Each of the values added to an array is placed in a separate slot and given an index number. Computers index from 0 therefore the 3rd array would be index 2.

  • Properties and Methods


    • Property is meta information about the object
    • Methods is the function that belongs to the object "Think of methods as small programs that we can use to do things to objects."
